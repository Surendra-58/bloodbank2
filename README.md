# bloodbank
# Blood Bank Management System

A web application designed to streamline blood bank management, including donor registration, blood requests, and inventory tracking.

## Features

- **Donor Management**: Register and manage donor details.
- **Inventory Tracking**: Keep track of available blood units by type.
- **Request Handling**: Manage blood requests efficiently.
- **User Authentication**: Secure login for users and administrators.

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Surendra-58/bloodbank2.git
   cd bloodbank2
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add the following:
   ```
   PORT=3000
   MONGO_URI=your_mongodb_connection_string
   ```

4. Start the server:
   ```bash
   npm start
   ```

5. Open the application in your browser:
   ```
   http://localhost:3000
   ```

## Usage

- Register as a donor or log in as an administrator.
- Add and manage blood inventory.
- Handle blood requests and track their status.
