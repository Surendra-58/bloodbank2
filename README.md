This blood bank management system is created in django, javascript, postgresql, cloudinary.

Features: 
1) Admin : manages everything.
2) Donor : They can accept blood request comming from admin.
3) Hospital : They can request blood in case of urgency in hospital to the admin.

There are lots of features including blog, all admin, donor, hospital can post blogs. 

Real time request tracking,inventory tracking and many more.
